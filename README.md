# Wook2Portfolio
문동욱의 포트폴리오

- HTML, CSS 및 JavaScript를 사용하는 반응형 웹사이트입니다.

- 스크롤 할 때 애니메이션을 포함합니다.

- 아이콘 소스는 'Boxicons' 사이트에서 사용하였습니다.

- 스크롤 관련 스크립트는 'Scrollreveal' 사이트에서 사용하였습니다.

- 디자인 편집은 Adobe 대신 Affinity 프로그램을 사용하였습니다.
